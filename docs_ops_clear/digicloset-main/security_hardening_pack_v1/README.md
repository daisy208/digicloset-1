# Security Hardening Pack v1.0

This pack contains templates, configuration files, scripts, and policies to harden the security posture of your application and infrastructure. Use these as drop-in templates and adapt to your environment.

**Contents overview:**
- CI security workflows (SAST, SCA, SBOM, secret scanning)
- Kubernetes security (Gatekeeper/OPA Rego policies, PodSecurity admission recommendations)
- Secrets management (HashiCorp Vault example)
- Runtime protection and WAF (ModSecurity sample)
- Monitoring & alerting for security events
- Incident response and forensics playbooks
- Audit & compliance helpers (SBOM, license checks)
- Pentest checklist and threat model template
