# DEPRECATED

This folder contains an older or experimental version of the backend services for the Digicloset project. It was created as part of initial or alternative upgrade pack implementations.

## Why it exists
This folder was used for early development, experimentation, or as a legacy implementation of backend services.

## What replaces it
The canonical and production-ready backend is now located in `digicloset-upgrade-pack/backend/` and `digicloset-upgrade-pack-complete/backend/`.

## Deletion/Archival
This folder should be archived for historical reference and not used for new development. Deletion is possible if no legacy dependencies exist.
